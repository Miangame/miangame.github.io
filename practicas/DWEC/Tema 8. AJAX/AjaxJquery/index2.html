Miguel Ángel Gavilán Merino - Primer Ejercicios AJAX

  1. Indica el significado de las constantes del objeto XMLHttpRequest:

    1. UNSENT: El método open() no ha sido llamado aún, código 0
    2. OPENED: open() ha sido llamado, código 1
    3. HEADERS_RECEIVED: se ha llamado al método send(), cabeceras y estado están disponibles, código 2
    4. LOADING: Descargando la respuesta del servidor, código 3
    5. DONE: La petición se ha completado, código 4

  2. Indica la forma de crear un objeto XMLHttpRequest hoy en día:

    La forma de crear un objeto es con let variable = new XMLHttpRequest();

  3. Métodos manejadores de eventos. Indica su utilidad.

    1. onreadystatechange: La funcion callback es llamada cuando el atributo readyState cambia.

  4. Respuestas del servidor (aparte de 200)

    1. 200: OK, cuando todo esta correcto.
    2. 403: Forbidden, cuando no hay permisos para acceder a ese recurso.
    3. 404: Page not found, cuando el recurso al que intentamos acceder no existe.

  5. Indica las distintas respuestas del servidor.

    1. 1xx: Respuestas informativas
    2. 2xx: Peticiones correctas
    3. 3xx: Redirecciones
    4. 4xx: Errores del cliente
    5. 5xx: Errores de servidor

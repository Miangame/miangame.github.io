3. Indica la función que comprueba si un objeto es o no un Array. Demuestra su uso mediante un ejemplo.

Para comprobar si un objeto es o no un Array necesitamos usar la función isArray(). Esta función nos devolverá true en caso de ser un objeto de tipo Array o false en caso de no serlo.

Ejemplo:
let coches = ["renault", "mercedes", "seat", "fiat"];
let cadena = new String("Hola a todos");

("¿coches es un array? : " + (Array.isArray(coches)));
("¿cadena es un array? : " + (Array.isArray(cadena)));

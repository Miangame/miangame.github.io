1. ¿Cuántos constructores tiene el objeto predefinido Date()? Explícalos mediante ejemplos.

EL objeto predefinido Date() tiene 4 constructores

  • let d = new Date(); - Muestra la fecha actual con el horario GMT +0
  • let d = new Date(milisegundos); - Crea una fecha según los milisegundos pasados por parámetros. Esta cantidad de milisegundos se le suma a una fecha inicial (año 1970) y se calcula la nueva fecha para mostrarnos una actual. En nuestro caso hemos hecho: let dateMs = new Date(Date.now()); Date.now() devuelve el momento actual en milisegundos
  • let d = new Date(cadenaConFecha); - Crea una fecha según la cadena dada. En dicha cadena se especifica primero el mes(en inglés), seguido del día, año, horas, minutos y segundos. En nuestro caso es: let dateString = new Date("11/05/2017");
  • let d = new Date(año, mes, dia, horas, minutos, segundos, milisegundos); - En este último caso, le pasamos los argumentos uno a uno para crear una fecha con dichos valores. Ejemplo: dateCompleto = new Date(2017, 11, 5, 0, 0, 0, 0);

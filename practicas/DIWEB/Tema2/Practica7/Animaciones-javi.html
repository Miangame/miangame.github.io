Usabilidad animaciones

Las animaciones son usadas para como su nombre indica, animar cualquiera de los elementos del árbol DOM que se le asigne dicha animación.

Ejemplo:
@keyframes identifier {
0% { top: 0; }
50% { top: 30px; left: 20px; }
50% { top: 10px; }
100% { top: 0; }
}

Declaración de animaciones

Las animaciones se declaran mediante la propiedad @keyframes y mediante llaves({}), se engloban las distintas propiedades.
Hay dos formas de declarar los @keyframes:

1. Dentro de las llaves se colocan las propiedad “from” y “to”. De forma que (From) englobará las acciones que se realizaran primero. Y la propiedad (to) englobará las acciones que se realizaran al final. Osea después del from.
Ejemplo:

@keyframes identifier {
from { top: 0; }
to { top: 30px; left: 20px; }
}

2. Dentro de las llaves se colocan porcentajes, indicando la cantidad proporcinal de la animación, desde 0%(inicio) a 100%(fin).
Ejemplo:

@keyframes identifier {
0% { top: 0; }
50% { top: 20px; left: 20px; }
100% { top: 40px; left: 40px; }
}

Propiedad animation

Es una propiedad para invocar animaciones declaradas previamente.
Los valores que admite (animation) son:
*animation: <nombre> <duración(segundos)> (opcional)<infinite> (opcional)<velocidad>;
(Se le pueden añadir muchos mas valores pero yo he indicado los mas usados)

Ejemplo:
div{ animation: pulsacion 5s infinite ease-out; }
@keyframes pulsacion {
0% { width: 50px; height:50px;}
50% { width: 100px; height:100px; }
100% { width: 50px; height:50px; }
}

Sub-propiedades

Son aquellas que aplican comportamiento de la animación que se invoca.
Hemos visto anteriormente que la animación se puede invocar indicando todos los valores seguidos.
-Por ejemplo así:

div{
animation: pulsacion 5s infinite ease-out;
}

Pero las sub-propiedades nos permiten dividir la invocación de la animación en partes, ya que cada sub-propiedad se encargará de una función.


Sub-propiedades:

div{
animation-name: pulsacion;
animation-duration: 5s;
animation-iteration-count: infinite;
animation-timing-function: ease;
...
}

En el ejemplo he puesto las sub-propiedades mas usadas. Pero mas adelante explocaré otras mas, que nos pueden ser útiles.

Sub-propiedad animation-name

Es necesario usarla para aplicar la animación que debe estar declarada(creada) previamente.
Como su nombre indica admite el nombre de la animación. Tenemos que tener en cuenta que solo con el nombre no nos funcionará la animación. Por lo que tendrás que leer las siguientes sub-propiedades para poder invocar combinarlas.

Ejemplo:

div{ animation-name: pulsacion; }
@keyframes pulsacion {
0% { width: 50px; height:50px;}
50% { width: 100px; height:100px; }
100% { width: 50px; height:50px; }
}

Sub-propiedad animation-duration

Es usada para indicar la duración de la animación. Se expresa en segundos.Tenemos que tener en cuenta que solo con la duración no nos funcionará la animación. Por lo que tendrás que leer las siguientes sub-propiedades para poder invocar combinarlas.

Ejemplo:

div{
animation-name: efectoMargin;
animation-duration:3s;
animation-iteration-count: infinite;
}
@keyframes efectoMargin{
0%{
margin-left: 0px;
}
50%{
margin-left: 100px;
}
100%{
margin-left: 0px;
} }
